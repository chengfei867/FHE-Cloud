# FHE-cloud
KeyGender用于密钥生成，在本地部署
直接在项目目录执行
#go build
#.\KeyGender.exe（linux系统没有.exe后缀）
端口为8080

FHE-cloud提供云计算，可以部署到远程服务器
直接在项目目录执行
#go build
#.\FHE_cloud.exe（linux系统没有.exe后缀）
端口为9090

完整目录结构如下，缺少的请手动创建
